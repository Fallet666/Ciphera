# Инструкция для семьи

## Начало работы

### 1. Открыть мессенджер

**В браузере:**
- Откройте https://chat.ru.potapovotrap.online

**На телефоне (Android):**
- Установите **Element X** из Google Play или F-Droid
- Или установите **FluffyChat** из Google Play, F-Droid или APK

**На компьютере:**
- Element Desktop: https://element.io/download
- Или используйте веб-версию

### 2. Регистрация

1. Нажмите **Create account**
2. Введите выданный вам **registration token** (например: `family-invite-mom`)
3. Придумайте **длинный пароль** (минимум 12 символов)
4. Нажмите **Register**

### 3. Сохранить ключ восстановления

После регистрации появится окно **Secure Backup**:

1. Нажмите **Set up Secure Backup**
2. Выберите **Generate a Security Key**
3. **Сохраните recovery key:**
   - Запишите на бумагу
   - Сохраните в парольном менеджере
   - Запишите на флешку

**Важно:** без recovery key вы не сможете восстановить доступ к зашифрованным сообщениям, если потеряете все устройства.

### 4. Добавить контакты

1. Нажмите **+** или **Start chat**
2. Введите имя пользователя: `@lesha:matrix.ru.potapovotrap.online`
3. Нажмите **Go**

### 5. Проверить шифрование

В каждом чате должен быть значок **замка** (encryption enabled).

Проверьте устройства собеседников:
1. Откройте профиль пользователя
2. Нажмите **Verify**
3. Сравните эмодзи или числа (SAS verification)

## Использование

### Отправка сообщений

- Текст: просто напишите и нажмите Enter
- Фото/видео: нажмите **скрепку** → выберите файл
- Голосовое: нажмите **микрофон** → запишите
- Файлы: нажмите **скрепку** → выберите файл

Лимит: 200 МБ на файл.

### Звонки

1. Откройте чат
2. Нажмите **телефон** (аудио) или **камера** (видео)

Для работы звонков нужен TURN-сервер (уже настроен).

### Групповые чаты

1. Нажмите **+** → **New Room**
2. Добавьте участников
3. Включите **Enable encryption** (если не включено автоматически)

## Безопасность

### Что можно

- Проверять устройства собеседников через verification
- Использовать длинные пароли
- Хранить recovery key в безопасном месте
- Сообщать админу о потере устройства

### Что нельзя

- Пересылать recovery key кому-либо
- Логиниться на чужих устройствах
- Использовать простые пароли
- Оставлять аккаунт открытым на общих устройствах

### Если потеряли телефон

1. **Немедленно** сообщите админу
2. Админ удалит устройство из системы
3. На новом устройстве войдите заново
4. Подтвердите новое устройство через verification

## Устранение проблем

### Не приходят уведомления

- Проверьте настройки уведомлений в Element
- На Android: разрешите уведомления в настройках системы
- На iOS: включите push-уведомления

### Не работают звонки

- Проверьте интернет-соединение
- Попробуйте другой браузер (Chrome, Firefox)
- Проверьте, что TURN-сервер доступен

### Не могу войти

- Проверьте правильность пароля
- Проверьте, что вводите правильный сервер: `matrix.ru.potapovotrap.online`
- Если забыли пароль — нужен recovery key для восстановления
